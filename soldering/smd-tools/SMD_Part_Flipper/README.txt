# Summary

SMD part flipper inspired by [Fristch Flipping Station](https://www.fritsch-smt.de/en/manual-pick-place/features/flipping-station-for-loose-components/).

Parts:

* 2x 3d printed flip plate
* 2x 50mm square x 5mm thick [ESD foam](https://www.banggood.com/200x200mm-ESD-Antistatic-Pin-Insertion-High-Density-Foam-3510mm-p-1114492.html)
* 2x M3x12mm bolt
* 2x M3 nut
* Double sided tape
