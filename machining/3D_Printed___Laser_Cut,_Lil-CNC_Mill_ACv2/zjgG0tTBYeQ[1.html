Published on Jun 9, 2015

This video shows a great little 3d printed, laser cut, and Open Source 3 axis CNC mill engraving a "hello world" test image into acrylic.

The video shows the entire tool path, but has been speed up to 2.5x speed as this is a ~20min etch job.

I was able to build the mill fairly quickly once all the ordered parts arrived, and I had finished printing the plastic parts.
Over all, for a hobby grade mill this machine is extremely capable, even with fine detail.

The square border around the logo is 40x40mm, and the text is at its tallest ~3.5mm tall. The cutting bit used was a 25deg, 0.3mm flat tip engraving bit.

This Open Source build is very well documented at.
http://www.instructables.com/id/3D-pr...

Very happy with how this project turned out.
